

# SimScreenSaver


ただのスクリーンセーバーです。
ホントにただのスクリーンセーバーです.

![](demo/demo_v0.2.0.jpg)

## 機能

機能はたったのこれだけ

・機能 1: アイコンを表示			(v0.1.0)
・機能 2: アイコンを定期的に変更	(v0.1.0)
・機能 3: CPU使用率を表示			(v0.2.0)
・機能 4: メモリ使用率を表示		(未実装)
