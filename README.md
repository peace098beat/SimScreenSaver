

# SimScreenSaver


ただのスクリーンセーバーです。ホントにただのスクリーンセーバーです.


## 機能

機能はたったのこれだけ

・機能 1: アイコンを表示			(v0.1.0)
・機能 2: アイコンを定期的に変更	(v0.1.0)
・機能 3: CPU, メモリ使用率を表示	(未実装)
